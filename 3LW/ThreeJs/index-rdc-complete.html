File completo

Questo file invece è il file completo vero e proprio. In questo caso il "sistema" andrà a cercare una telecamera esistente, e verrà considerata l'aniamazione generata con Cinema 4D (riga 206). Bisogna tenere conto che sarà possibile generare animazioni solo con 2 keyframe, uno all'inizio e uno alla fine della timeline. Il file funziona sia con lo scroll che con lo swipe su mobile.

Testato con export da Cinema r17 - Collada versione 1.4

  • Rispetto ad un normale progetto Three.js ha bisogno di questi file aggiuntivi: ColladaLoader, Detector, inertiaScroll
  • Generare modello in Cinema 4D
  • Generare texture con Body Paint, applicare la texture come jpg e tenersi il sorgente in PSD
  • Generare l'eventuale alpha dalla stessa sorgente PSD
  • Caricare il file nella riga 36 (settare bene la path dal file che carica il JS)
  • Inserire gli elementi che non hanno il canale alpha nei "forbidden names", alla riga 508
  • Settare la path del canale alpha (se necessario), alla riga 556
  • Settare la durata dell'animazione alla riga 208
